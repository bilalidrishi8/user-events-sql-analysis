# 📊 User Event Analytics using SQL

## 📌 Project Overview

This project analyzes user event data using SQL to uncover valuable business insights. The dataset contains user interactions such as page views, clicks, sign-ups, purchases, and traffic sources. Through SQL queries, the project explores user behavior, engagement, revenue trends, and conversion metrics.

---

## 🎯 Objectives

* Perform data exploration and cleaning
* Analyze user behavior and engagement
* Measure purchase performance
* Identify top traffic sources
* Generate business insights using SQL
* Practice beginner to advanced SQL concepts

---

## 🗂️ Dataset Information

| Column         | Description                                         |
| -------------- | --------------------------------------------------- |
| user_id        | Unique identifier for each user                     |
| event_type     | Type of event (View, Click, Signup, Purchase, etc.) |
| event_time     | Date and time of the event                          |
| amount         | Purchase amount (if applicable)                     |
| traffic_source | Source from which the user arrived                  |
